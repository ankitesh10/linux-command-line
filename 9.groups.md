# Groups

```text
drwxr-x--- 1 ubuntu ubuntu 4.0K May 14 15:48 ubuntu
```

- `d` means directory.
- The first `rwx` is for the user: read, write, execute.
- The second `r-x` is for the group: read and execute only.
- The third `---` is for everyone else: no permission.

## Change Owner

```sh
sudo chown ubuntu:ubuntu <directory/file>
```

## Change Mode

```sh
sudo chmod u=rw,g=rw,o=rw <file/directory>
```
