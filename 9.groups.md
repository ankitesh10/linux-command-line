# Groups

```text
drwxr-x--- 1 ubuntu ubuntu 4.0K May 14 15:48 ubuntu
```

- `d` means directory.
- The first `rwx` is for the user: read, write, execute.
- The second `r-x` is for the group: read and execute only.
- The third `---` is for everyone else: no permission.

## Change Owner

```sh
sudo chown ubuntu:ubuntu <directory/file>
```

## Change Mode

```sh
sudo chmod u=rw,g=rw,o=rw <file/directory>
```

```sh
sudo chmod 777 <file/directory>
```

- first 7 is user, second group and third other.
- r is 2<sup>0</sup> = 1
- 2 is 2<sup>1</sup> = 2
- x is 2<sup>2</sup> = 4

### Add certain permision

```sh
sudo chmod u+w,g+w,o+w filename
sudo chmod a+w filename         # Add write permission for all
```
