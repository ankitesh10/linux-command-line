# Groups

```
drwxr-x--- 1 ubuntu ubuntu 4.0K May 14 15:48 ubuntu
```

- d mean directory
- first rwx is for user, read write execute
- second r-x is for group, read and execute only
- third --- is for rest, no permission

## Change owner

```
sudo chown ubuntu:ubuntu <directory/file>
```

## Change Mod

```
sudo chmod u=rw,g=rw,o=rw <file/directory>
```
