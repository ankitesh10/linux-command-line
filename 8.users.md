# Users

## Current User

```sh
whoami
```

`whoami` prints the current user name.

## List Users

```sh
cat /etc/passwd
```

## Superuser

```sh
sudo
```

`sudo` means superuser do. The superuser is `root`.

## Add a User

```sh
sudo useradd ankitesh
sudo passwd ankitesh
```

## Switch User

```sh
su ankitesh
```

## Add a User to the Sudo Group

```sh
sudo usermod -aG sudo ankitesh
```
