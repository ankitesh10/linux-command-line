# Wildcards

## Create Files with Brace Expansion

```sh
touch file{1,2,3,4}.txt
touch file{1..30}.text
touch file{a..z}.text
touch file\?.txt
```

- `touch file{1,2,3,4}.txt` expands to `touch file1.txt file2.txt file3.txt file4.txt`.
- `touch file{1..30}.text` expands from 1 to 30.
- `touch file{a..z}.text` expands from `a` to `z`.
- `\` escapes the next character.

## Match Files with Wildcards

```sh
ls file-*     # match everything after file-
ls file?.txt  # match exactly 1 character
ls file??.txt # match exactly 2 characters
```

## Echo Ranges

```sh
echo {a..z}
echo {a..z..10}
echo {0..100..10}
echo {0..10}{a..z}
```

- `{a..z..10}` skips every 10.
- `{0..10}{a..z}` combines numeric and letter ranges.
