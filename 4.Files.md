# Files

## Reading Files

```sh
less <file>       # read a large file in a Vim-like view
man less          # manual page for less
cat <file>        # print a file; not ideal for large files
tail <file>       # last 10 lines
head <file>       # first 10 lines
tail -n 3 <file>  # last 3 lines
tail -f <file>    # follow a file in real time as lines are added
```

## Create Multiple Folders

```sh
mkdir -p /cd/home/aa
```

## Tar Files

- A tar file is a file format used to combine a group of files.
- It globs a bunch of files together so they can be extracted later.
- It is also sometimes called a tarball.

```sh
tar -cf archive.tar <file_names>
tar -zcf archive.tar.gz <file_names>
tar -xzf archive.tar.gz
tar -xf archive.tar -C <folder_name>
```

- `x` is the flag for extract.
- `-C` extracts into a folder.
