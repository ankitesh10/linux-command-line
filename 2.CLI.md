# What is Bash?

- You are using a shell right now, and that shell is almost certainly called Bash (it definitely is unless you changed something),
  the Bourne Again Shell, which is making fun of the Bourne shell that Bash replaced.
- Bash is by far the most common shell and is over 30 years old.
- A shell is how you interact with your operating system.
- Bash is a command-line shell and scripting language used to interact with Unix-like systems.
- REPL means Read Evaluate Print Loop. It's basically an interactive way of programming where you're writing
  one line of code at a time, feeding data in and out of little programs.

## Commands

```sh
pwd              # present working directory
ls               # list files
cd               # change directory
echo hi          # prints the parameter
which ls         # prints the path of a program
ls --help        # get help for ls
```

```text
~ -> home directory
/ -> root directory
```

## Flags

```sh
ls --help        # ask for help
ls -l            # long listing format
ls -la           # include hidden files
touch .hidden-file
```

There are two styles of flags:

```sh
ls -a            # short flag
ls --all         # long flag
ls -al           # combined short flags
ls -la           # same combined short flags in a different order
```

More examples:

```sh
ls -lsah
ls --ignore=bin
ls --ignore=bin --ignore=boot
```

## Bash History

```sh
tail ~/.bash_history
!!
clear
```

- `ctrl+r` searches Bash history.
- `tail ~/.bash_history` prints the last 10 lines of Bash history.
- Bash history is written after the user logs out.
- `!!` runs the last command.
- `clear` or `ctrl+l` clears the screen.

## Shortcuts

- `CTRL + A` takes you to the beginning of the line.
- `CTRL + E` takes you to the end of the line.
- `CTRL + K` yanks everything after the cursor.
- `CTRL + U` yanks everything before the cursor.
- `CTRL + Y` pastes everything you yanked. It does not go into your system clipboard.
- `CTRL + L` clears the screen.
- `CTRL + R` reverse searches through history.
